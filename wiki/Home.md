# InterServerPortal Wiki

**InterServerPortal** lets a player standing on a dedicated Valheim server step
through a portal and switch directly to **their own local single-player world** —
and step back to return to the server. It replaces the manual *log out → main menu
→ pick world → load → reconnect* cycle with a single portal interaction. The same
mod also adds **same-world portal networks** that mesh all of your portals so you
can hop between any of them.

It uses **vanilla assets only** (`portal_wood` and `portal_stone`) and leaves
portals you don't configure completely untouched.

> ### The one hard rule
> Valheim binds you to one world per session. There is no way to be in two worlds
> at once or to switch without a loading screen. This mod does **not** fight that —
> it removes the *menu navigation*, not the load. **Every transition costs exactly
> one loading screen.**

## At a glance

| You want to… | Do this |
|---|---|
| Go to your own world from a server | Flag a portal **inter-server**, add a destination, walk through |
| Come back to the server | Walk through a portal in your local world → **Return to origin server** |
| Hop between your own portals in one world | Set portals to **Network** link mode, walk through one |
| Require a code to use a portal | Set an entry code in the config panel (**L.Shift + Use**) |
| Get a Discord ping when you cross over | Set `Discord/WebhookUrl` in the config |
